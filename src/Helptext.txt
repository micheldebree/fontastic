Important Operations:

Menus:
------

  Enter/Exit Menu's  : ESC
  Move selection-bar : CURSOR KEYS
  Make selection     : SPACE or ENTER

Selecting a char to edit from the Char-Map:
-------------------------------------------

  Move cursor         : CURSOR KEYS
  Select char to edit : SPACE or ENTER
  Enter main menu     : ESC (Quit, File, Help)

Editing a char:
---------------

  Move cursor         : CURSOR KEYS
  Flip Pixel          : SPACE or ENTER
  Enter edit menu     : ESC  (Quit, Scroll, Flip, Clear, Invert, Read, Write)

Selecting a filename:
---------------------

  Choose name    : Same as in menus
  Enter name     : Choose <name> or empty name, type name and press ENTER
  See more files : Select <more> if present
  Exit           : ESC


Menu selections:
----------------

 Main Menu : Quit  : Exits FONTASTIC (gives warning when font not saved)
             File  : Invokes Files Menu (load & save fonts)
             Help  : You allready found that out...

 File Menu : Exit      : Exits the File Menu (same as ESC)
             Load Font : Loads a font from disk. Read "Selecting a filename"
             Save Font : Saves a font to disk. Read "Selecting a filename"
             Dos Shell : Enters the Dos-shell. Type EXIT to return to
                         FONTASTIC. N.B. : If there is not enough memory
                         available to run a copy of COMMAND.COM, FONTASTIC is
                         immediately returned to...

 Help : Scroll Up/Down : CURSOR KEYS
      : Exit Help      : ESC

 Edit Menu : Quit Edit : Exits the edit mode
             Scroll    : Scrolls the char in the four directions by using the
                         CURSOR KEYS. ESC to stop scrolling
             Flip      : Flips the char in horizontally or vertically by
                         using the CURSOR KEYS. ESC to stop flipping
             Clear     : Clears the char
             Invert    : Inverts the char
             Read      : Overwrites the char with data read from buffer
             Write     : Overwrites buffer with the char data
